# Npm package boilerplate for typescript

Boilerplate includes support for
- [Typescript transpiled by esbuild](https://esbuild.github.io/)
- [Eslint](https://eslint.org/blog/2022/01/eslint-v8.8.0-released/#main)
- [Prettier](https://prettier.io/docs/en/index.html)
- [Jest](https://jestjs.io/docs/getting-started)
- [Code coverage](https://www.npmjs.com/package/codecov)
